# Security Policy

## 支持的版本

由于这是个人的配置文件 (Dotfiles) 我通常只维护最新版本的配置  
如果你在使用旧版本时遇到问题, 建议先同步到最新的 main 分支

| 版本 | 是否支持 |
| :---: | :---: |
| Latest (main) | ✅ Yes |
| < Latest | ❌ No |

## 报告安全问题
**​请不要直接在公共 Issue 中报告安全漏洞**
1. **硬编码的机密信息**: 比如我不小心提交了明文密码, API 令牌, 私钥等
2. **脚本安全漏洞**: 安装脚本中存在可能导致系统被恶意劫持的漏洞

​请通过以下方式私下告知我:
* **发送邮件**: jyin1221@gmail.com
* **GitHub 私有漏洞报告**: 如果本仓库开启了此功能, 请直接提交

​我会在收到通知后 72 小时内确认问题, 并尽快通过 git push --force 或其他方式清理敏感历史记录

## 安全建议
如果你准备 Fork 或使用我的配置, 请注意:
- **脚本审查**: 在你的机器上运行任何 ./setup.sh 脚本前，请务必先阅读源码
